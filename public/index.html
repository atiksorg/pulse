Эндпоинты

Базовый URL: https://events.atiks.org. CORS открыт — обращайтесь напрямую из браузера, мобильного приложения или бэкенда.

МетодПутьОписание
GET/eЗапись одного события через query-параметры
POST/eЗапись одного события через JSON-тело
POST/e/batchПакетная запись массива событий (до 1000 за раз)
GET/sАгрегированная статистика: группировки и агрегации
POST/e/clearУдаление событий по src (и опционально type)
GET/exportЭкспорт «сырых» событий в CSV
GET/healthБыстрая проверка доступности сервиса
GET/statusРасширенный мониторинг: память, БД, конфигурация

Как отправить событие

Один и тот же вызов — на пяти языках. Замените src на своё имя источника.

Как получить статистику

GET /s считает события и агрегации на лету по всем таблицам сразу.

group
day, hour или field:название — группировка по значению произвольного поля payload.
agg
По умолчанию — count. Также sum:поле и avg:поле.
from / to
ISO-даты для ограничения диапазона: ?from=2026-07-01
type
Фильтр по конкретному типу события внутри источника.
# количество событий типа purchase по дням за последние 30 дней
GET https://events.atiks.org/s?src=my_app&type=purchase&group=day&from=2026-06-06
